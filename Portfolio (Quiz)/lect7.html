•What is Bootstrap?

Bootstrap is a free and open-source front-end web framework for designing websites and web applications. It contains HTML, CSS, and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.



•How do you use Bootstrap containers?

Bootstrap container has 2 containers available which are .container and .container-fluid. Containers are used to pad the content inside of them. Containers in Bootstrap are a fundamental building block for creating a responsive design. Containers are used to wrap your content and apply padding and width restrictions on different devices.



•How do you use Bootstrap in Typography?

Bootstrap provides various classes to style text, such as headings, paragraphs, lists, and more.



•How do you use Bootstrap in Colors?

Bootstrap provides various classes to style background colors, such as `bg-primary`, `bg-success`, `bg-danger`, `bg-warning`, `bg-info`, and `bg-secondary`.
